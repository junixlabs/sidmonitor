# Project Brief: Sid monitoring

## Overview
Project để đọc file log từ Laravel project về tất cả request được gọi đến backend (log lưu trữ request ID, user, module, tags,... và các thông tin về HTTP request). Và log về tất cả request gọi ra third party API. Dự án sẽ cung cấp dashboard để theo dõi tình trạng các service nội bộ và các service sử dụng bên ngoài.

## Tech Stack
- **Type:** Fullstack (backend đơn giản - transform data cho frontend)
- **Frontend:** React
- **Backend:** Python
- **Database:** ClickHouse
- **Authentication:** Basic Auth

## MVP Goals
Dashboard để monitor các API theo trạng thái request, endpoint, module, user.

## Core Features
1. Dashboard tổng quan
2. Filter theo status/endpoint/module/user
3. Chart thống kê request
4. Log viewer chi tiết
