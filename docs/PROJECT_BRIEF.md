# Project Brief: SidMonitor

## Overview
Log monitoring system for Laravel projects. Provides a library and monitoring dashboard to help users integrate into API systems for logging:
- **Inbound requests:** path, module, method, status_code, user, etc.
- **Outbound requests:** third-party API calls with path, method, status_code, service

**Goal:** Log and monitor API services via HTTP requests

## Tech Stack
- **Type:** Fullstack
- **Frontend:** React 18 + TypeScript + Vite + TailwindCSS
- **Backend:** Python FastAPI
- **Database:** ClickHouse
- **Authentication:** Basic Auth
- **Client Library:** Laravel Package

## MVP Goals
- Test with 1-2 Laravel projects
- Validate logging and monitoring workflow
- Gather feedback before scaling

## Core Features (MVP)
1. **Laravel Library** - Log inbound/outbound requests from Laravel apps
2. **Dashboard Overview** - Total requests, error rate, response time metrics
3. **Filter Logs** - By status, endpoint, module, user
4. **Log Viewer** - Detailed view of individual requests
5. **Ingest API** - Receive logs from Laravel library
